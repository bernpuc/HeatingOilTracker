# Sample Data

This folder contains sample data for demonstrating Heating Oil Tracker's features.

## demo-deliveries.csv

A synthetic dataset with ~2.5 years of fictional oil deliveries (2022-2024). Use this to explore the app's features without entering real data.

**To import:**
1. Open Heating Oil Tracker
2. Go to **Deliveries**
3. Click **Import CSV**
4. Select `demo-deliveries.csv`

**To remove demo data:**
- Delete individual deliveries from the Deliveries view, or
- Delete the data file at `%APPDATA%\HeatingOilTracker\data.json` to start fresh

**Note:** This is synthetic data with round numbers and "Demo data" notes. It does not represent any real household's consumption.
